This function is called only if HeadConfig.isSearchAvailable is enabled, it's used to search items in the module.

This section can be filtered if searchFilters is declared.

Return

A PagingData flow for the search result.

Parameters

query

The query to search

bypassModel

Cloudflare bypass information extracted by the Main app, if your module doesn't require a bypass you can disable it in HeadConfig.bypassBehavior.

filters

Requested filters by the user.
